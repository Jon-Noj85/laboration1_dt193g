Om laborationen

Denna laboration går ut på att utforska olika CSS-ramverk. Ramverken används för att göra det smidigt att designa webbplatser och applikationer. Ramverk kan konsumeras på lite olika sätt, såsom CDN, rena filer eller installation med package manager.

Den andra delen av laborationen går ut på att publicering av webbplatsen inte ska ske via FTP (File Transfer Protocol) utan på annat sätt.

Valda ramverk

Jag valde Bootstrap och Foundation, som är två av de flitigast använda ramverken. Bootstrap är open-source och underhålls via GitHub. På Bootstraps webbplats finns information om hur man konsumerar och använder ramverket.
