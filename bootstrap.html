Bootstrap

Bootstrap är ett CSS-ramverk som initialt skapades av utvecklare på Twitter men numera underhålls genom open-source på GitHub. På Bootstraps webbplats finns en stor mängd dokumentation om hur man konsumerar Bootstrap.

Bootstrap är det vanligaste CSS-ramverket, och därför ville jag prova det. Statistik över olika CSS-ramverk. Bootstrap går att konsumeras på olika sätt: på denna sida genom CDN. Både CSS och JavaScript hämtas genom CDN. Detta gör Bootstrap enkelt att använda och gör sidan snabb.

Jag tycker att Boostrap fungerar väldigt smidigt, då det är enkelt att hitta dokumentation över användning samt att det går att modifiera element genom olika klasser på dessa. Till exempel genom att ändra färgteman, och grid-layout kan man göra ändringar som gör sidan lite unik. Trots dessa ändringar är det ganska tydligt att man använts sig av just Bootstrap.

Framförallt blev det enkelt att på front end-sidan verifiera formulärets inmatningsfält med hjälp av JQuery. Dokumentation över hur man validerar formulär.
